# Electrochem ML Layers

A collection of physics-inspired neural network layers for electrochemical systems.

This repository provides reusable PyTorch modules that embed electrochemical kinetics directly into machine learning models.

## Currently Available Layers

### Butler–Volmer Layer (`BV_Layer`)

A differentiable implementation of Butler–Volmer kinetics as described in https://chemrxiv.org/doi/full/10.26434/chemrxiv-2025-kxg13/v2
